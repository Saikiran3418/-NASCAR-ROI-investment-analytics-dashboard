# 📊 ROI Investment Analytics Dashboard

## 🔍 Overview

End-to-end analytics project using SQL Server, Python (Google Colab), and Power BI to evaluate sponsorship ROI and optimize investment decisions.

---

## 🛠 Tech Stack

* SQL Server (Data extraction)
* Python (Data cleaning & transformation)
* Power BI (Visualization & insights)

---

## 📊 Dashboard Features

### 🔹 ROI by Driver

* Identifies top and underperforming drivers

### 🔹 Risk vs Return Analysis

* Shows relationship between investment and ROI
* Highlights inefficient spending

### 🔹 Recommendations

* Invest More → High ROI drivers
* Avoid Investment → High cost, low ROI
* Optimize → Moderate performers

---

## 🧠 Key Insight

High investment does NOT guarantee high ROI. Several drivers show inefficient spending patterns.

---

## 💡 Business Impact

Helps stakeholders:

* Reallocate budget effectively
* Improve ROI performance
* Make data-driven investment decisions
